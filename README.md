# Contact Manager Python

This is a simple Python project for managing contacts.

## Project Files

- contact_manager.ipynb: The main Jupyter Notebook file.
- contacts.csv: The main contacts data file.
- contacts.xlsx: Excel version of the contacts data.
- gmail_contacts.csv: Exported Gmail contacts.
- duplicate_report.txt: Report for possible duplicate contacts.
- README.md.docx: Word document for the project readme.

## Features

- Load contacts from a CSV file.
- Search contacts by name, email, company, or tag.
- Filter contacts by tag and email domain.
- Find contacts with or without phone numbers.
- Detect possible duplicate contacts.
- Export Gmail contacts to a CSV file.
- Export duplicate contacts report to a text file.

## How to Use

1. Open `contact_manager.ipynb` in Jupyter Notebook.
2. Run the cells from top to bottom.
3. The notebook will read `contacts.csv`.
4. The notebook will create or update:
   - `gmail_contacts.csv`
   - `duplicate_report.txt`
